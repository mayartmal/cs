Метод Ньютона

x = sqrt(y)
y = x^2
f(x) = x^2 - y

Позволяет найти корень целого числа
y - число
x - оценка

>>> y = 2.0
>>> x = y
>>> eps = 1e-5 * x
>>> while (x - y / x) > eps:
...     x = (x + y / x) / 2.0
>>> x
1.4142156862745097
