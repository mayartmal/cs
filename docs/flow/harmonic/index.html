Гармонические числа

\sum_{i=1}^{n}\frac{1}{i}

>>> 1/1 + 1/2 + 1/3 + 1/4 + 1/5
2.283333333333333


>>> n = 5
>>> total = 0.0
>>> for i in range(1, n + 1):
...     total += 1.0 / i
>>> total
2.283333333333333
